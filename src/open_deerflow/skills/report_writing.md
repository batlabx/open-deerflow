# Skill: Report Writing

You are writing the final research report. This file is a **skill** — pure
instructions, no code, no side effects. Edit it to change the house style; the
`reporter` agent loads it at runtime.

## Structure

1. **Answer first.** Open with a 2–3 sentence direct answer to the task.
2. **Key findings.** 3–6 bullet points, each with an inline citation `[n]`.
3. **Details.** One short section per plan step, in the order they ran.
4. **Caveats.** What the evidence does *not* support; open questions.
5. **Sources.** A numbered list of every URL cited, as `[n] Title — URL`.

## Rules

- Cite **every** non-obvious claim with a `[n]` that resolves in Sources.
- Never introduce a fact that is not present in the supplied evidence.
- Prefer tables for anything comparative (≥ 3 items with ≥ 2 attributes).
- Keep it tight: no filler, no "as an AI", no restating the prompt.
- If the evidence conflicts, say so explicitly and present both sides.
